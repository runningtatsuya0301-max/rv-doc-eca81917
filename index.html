ケイデンス（ピッチ）の原則 ── なぜ「現状+5-10%」なのか
① ピッチを上げると接地が真下に寄る：同じペースのままピッチを上げる → 歩幅が自然に縮む → 足を前に放り出す距離が減る → 接地点が体の真下へ。踵から「ドスッ」のブレーキ衝撃が減る。※スピードは上げず、足だけ「速く・小さく」回すのがコツ。
② ケイデンスはシーソー：低すぎ(<170)→歩幅を前に稼いで脛・膝・腰へ／高すぎ・急増→蹴り・引き上げの頻度↑でふくらはぎ・アキレス・足部へ。だから最適は中庸＝現状から+5-10%・段階的に。「全員180spm」は廃止（鈴木さんFB）。
③ 速い選手のケイデンス一定は真似しない：エリートは高いピッチ基盤＋後方への推進でストライドを伸ばすから速くても安全。基盤のない市民ランナーが歩幅を前に伸ばして真似すると、前接地になって脛・膝に来る。

✅ 監修済

§7
SYMPTOM
膝が痛い （お皿周り/外側/お皿下/奥/離地時）
🔍
まず痛む「場所」で正体が変わる ── 同じ"膝痛"でも別の病気（先に見分ける）
・お皿の下の骨の出っぱり（成長期10〜15歳） → オスグッド：太もも前(四頭筋)が、まだ柔らかい脛骨粗面を毎歩引っぱって剥離・炎症。＝前ももの硬さ＋ジャンプ/ダッシュの反復。
・お皿のすぐ下の腱（大人） → 膝蓋腱炎(ジャンパー膝)：着地のたび膝蓋腱が遠心性に引き伸ばされ微細断裂が蓄積→変性。＝四頭筋優位・着地のブレーキ（Q2/Q3）。
・膝の内側（すね内側の上） → 鵞足炎(がそくえん)：膝が内に入る(ニーイン)・過回内で、内側を支える3つの腱の付着部が摩擦＆牽引で炎症。＝中殿筋の弱さ・過回内（Q1）。
・膝の奥・引っかかり・"水がたまる" → 半月板：荷重＋ねじれで挟まれ摩耗。水（関節水腫）は"中で炎症が起きてるサイン"＝結果であって原因ではない（裏に半月板・軟骨・滑膜炎が必ずある）。＝膝のねじれ・距離急増（Q1/Q5）。
・膝の外側 → 腸脛靭帯炎(ITBS) → #6タブへ。

SYMPTOM
脛が痛い（シンスプリント）
🔍
まず聞き分け：「内側」か「前面」か（意味がまるで違う）
・内側が痛い → シンスプ本体（後脛骨筋系）。張り → 骨膜炎 → 脛骨「内側」の疲労骨折という連続体に乗る。だから疲労骨折を警戒し毎週チェック。
・前面が張る → 前脛骨筋の筋疲労（別系統）。シンスプ本体（骨膜炎→疲労骨折ライン）には基本進まない。これは「接地が前すぎる＝オーバーストライド」のサイン＝今ピッチを直せば、進む前に止められる。

→ やめなくていい。 つま先で着こうとすると、今度はふくらはぎ・アキレス腱が悲鳴を上げる（衝撃の置き場所が移るだけ）。さらにつま先を上げる意識は前脛骨筋を余計に使って逆効果。やるのは「足を速く小さく回す（ピッチ+5-10%）」だけ。着く場所が体の真下になれば、かかと着地のままでも脛は楽になる。

SYMPTOM

🔍
「走ると(大転子・腸腰筋が)張る／座ると伸びて気持ちいい」のロジック
① なぜ走ると張る：腸腰筋(大腰筋＋腸骨筋)は腰椎・骨盤の前面 → 太腿の付け根(小転子)につき、脚を前に振り上げる(股関節を曲げる)主役。走りは毎歩この屈曲の連続→使いっぱなしで疲労・短縮して張る。とくにお尻(大殿筋)が弱い／オーバーストライド／反り腰(骨盤前傾)だと、後ろへ押し出せず"前で引き上げる"走りになり腸腰筋を酷使。大転子まわりは中殿筋・TFL・お尻外側＋腸脛靭帯が骨盤の横ブレを止め続け→外側が張る。
② なぜ座ると気持ちいい（ここが要注意）：座る＝股関節を曲げる姿勢。腸腰筋は実は"縮んでオフ"になる＝伸びてはいない。走行中ずっと緊張していた筋が解放されて脱力する「ゆるみ」を「伸びて気持ちいい」と感じている可能性が高い。一方、脚を組む/あぐら/お尻を伸ばす座り方は大転子まわり(お尻外側・TFL・梨状筋)が本当に伸びて気持ちいい。
③ 落とし穴：腸腰筋が硬い人は"座りっぱなし"がむしろ短縮を進める→次に走るとさらに引っ張られて張る悪循環。本当のケアは座って休むより、立って/うつ伏せで股関節を"伸ばす方向(伸展)"に伸ばす＝カウチストレッチ＋お尻起動(Q1/Q2の改善メニュー)。
